# Chapter 3 — AI for Personal Productivity

**TL;DR:** AI helps plan, organize and reduce stress.

## Uses
- Scheduling
- Emails
- Focus

## Practical prompts
```
Make a perfect daily schedule for me
```
```
Write a professional email asking for leave
```
```
Create a 3 hour deep focus plan
```

## Steps
1. Write your goal
2. Ask AI for a plan
3. Follow it
